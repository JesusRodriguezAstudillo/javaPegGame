In order to compile the program on the CSE machines use the line
	javac peg.java

Then run the program with the line
	java peg

The second command will automatically start the peg game and solve
it. To learn more about the code check the comments in the file.

Sample usage and output:

jr0463@cse05:~/4430/javaPrograms$ javac peg.java
jr0463@cse05:~/4430/javaPrograms$ java peg
